# Fedrick Region

Recruiting landing page (Next.js). **All app code is in [`web/`](./web/).**

## Deploy on Vercel

1. Connect this repo: https://github.com/SamuelPetrucci/fedrickReigion  
2. Set **Root Directory** to **`web`**  
3. Framework: **Next.js** (leave Output Directory empty)  
4. See [`web/DEPLOYMENT.md`](./web/DEPLOYMENT.md) for env vars and troubleshooting.
